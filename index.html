HOW TO PLAY
CHEST ROLLS
Each chest contains 3 random choices, a buff, heart or Points. Upgrades stack. Pick Wisely!
HEART — HEAL 1 HEART [MAX 5]
SWORD — BIGGER BLADE
SPEED — INCREASE SPEED
STEP — IMPROVE DODGE
MAGIC SHIELD — ABSORB HIT & BURSTS IN SHOCKWAVE
The world has secrets. Find them. (Try looking for breakable objects)
HALF HEART — HEALS 1/2 HEART
HEALTH POTION — HEALS 1 HEART
DESKTOP
ARROWS / WASD = MOVE
SPACE / LEFT CLICK = ATTACK
ENTER / E = INTERACT
ESC = PAUSE GAME
DODGE — SHIFT
MOBILE
SWIPE / DRAG = MOVE
TAP = ATTACK
E BUTTON — INTERACT
(bottom left)
DODGE BUTTON — DODGE
(bottom right)
TAP HUD — USE POTION
(top left)
Scoreboard is stored locally as a cookie. Clear cache to reset.

If you find the game too difficult, type "immagundam" for player-name to become invincible.
